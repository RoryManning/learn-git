Note: This project assumes you already have a Github account and that this repo has been forked to it. If not, do that first, and then come back.

First things First A review of what you've done so far

If you've gotten this far, you're doing great already! The first thing you did was fork this repository into your own Github account. That means you made your own copy of it, where you can make your own changes and upgrades without messing with the officially copy. Someday, you might want to make changes to the official copy, and that's great! Welcome to open source software. However, we're not up to that just yet, so lets take things

one step at a time. Once you had your own version of the site saved to your Github account, you cloneed it onto your computer. There is another action, pull, that is very similar, which gets new changes to the remote version of your project.

If you're following along with my in class, or if you've following a set up to-be-written instruction, you don't have to worry about this at all! We'll make sure you do the right thing the first time.
